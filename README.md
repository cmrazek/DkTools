# DkTools
Visual Studio Extension for WBDK

**Features:**
- Syntax highlighting
- Go to / peek references
- Quick-info on mouse hover
- Find all references
- Statement completion
- Function signature help
- Code snippets
- Automatic brace completion
- Code folding
- Brace matching
- WBDK compiler
- WBDK file / dictionary explorer
- Static code analysis
